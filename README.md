# 🎲 骰子挑戰遊戲 (Dice Dare)

一款派對用骰子挑戰遊戲，支援 Game Master 設定挑戰、分享連結給玩家擲骰。

## 玩法

### Game Master 模式
1. 點擊 **Game Master** 進入設定頁面
2. 編輯挑戰內容（預設已有 6 個挑戰）
3. 點擊 **🔗 產生分享連結** 把連結傳給其他玩家
4. 或點擊 **🎲 直接開始遊戲** 在同一裝置上玩

### Roller 模式
1. 透過 GM 分享的連結開啟遊戲
2. 點擊 **🎲 擲骰子！** 隨機抽取一個挑戰
3. 完成挑戰後可繼續擲骰
4. 所有挑戰完成後可點擊 **🔄 重置** 重新開始
